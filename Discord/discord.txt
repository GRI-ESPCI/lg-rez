	C'est le code général (toutes les fonctions qui interfaçaient Chatfuel, qui devront être modifiées) ; le code spécifique au bot est dans le dossier Discord


Le bot tourne en permanence sur le serveur alwaysdata. Pour publier son code, il faut d'abord le transférer : 
	scp -ro User=lg-rez Discord/*  ssh-lg-rez.alwaysdata.net:~/www/Discord
	}fQ%9J37a


Puis redémarrer le bot :
	ssh -l lg-rez ssh-lg-rez.alwaysdata.net
	}fQ%9J37a
	./reboot_bot.sh
