Le bot tourne en permanence sur le serveur alwaysdata. Pour publier son code, il faut d'abord le transférer :
	- Unix (après apt-get install sshpass):
		sshpass -p '}fQ%9J37a' scp -ro User=lg-rez Discord/* ssh-lg-rez.alwaysdata.net:~/www/Discord

	- Windows :
		scp -ro User=lg-rez Discord/* ssh-lg-rez.alwaysdata.net:~/www/Discord
		}fQ%9J37a


Puis redémarrer le bot :
	- Unix (après apt-get install sshpass):
		sshpass -p '}fQ%9J37a' ssh -l lg-rez ssh-lg-rez.alwaysdata.net ./reboot_bot.sh

	- Windows :
		ssh -l lg-rez ssh-lg-rez.alwaysdata.net ./reboot_bot.sh
		}fQ%9J37a


On peut bien sûr enchaîner le tout :
	sshpass -p '}fQ%9J37a' scp -ro User=lg-rez Discord/* ssh-lg-rez.alwaysdata.net:~/www/Discord && sshpass -p '}fQ%9J37a' ssh -l lg-rez ssh-lg-rez.alwaysdata.net ./reboot_bot.sh
